RStudio 살펴보기 - 1. 콘솔 창

콘솔 창은 코드를 실행하고 텍스트 결과물을 출력하는 창입니다.

1+1을 입력하고 Enter 키를 누르면 결과를 출력합니다.

명령프롬프트

  • >는 코드를 실행할 준비가 되었음을 의미합니다.

  • +는 앞서 실행한 명령문이 불완전했음을 의미합니다. (Esc로 탈출)

RStudio 살펴보기 - 2. 소스 창

소스 창은 코딩을 하는 공간이며, 코드를 저장한 파일을 스크립트라고 합니다.

[1]  TRUE  TRUE FALSE FALSE

# 조건을 만족하는 원소 자리에 TRUE를 준다.

Tidy Data (깔끔한 데이터)란?

“wide-form”이 아닌 “long-form” 형식의 데이터

  • 각 행이 관측점

  • 각 열이 변수

사람에게는 한 화면에 더 많은 데이터를 볼 수 있는 “wide-form”이 더 직관적입니다.

컴퓨터에게는 데이터베이스 형식에 가까운 “long-form”이 더 가독성이 좋습니다.

  • R의 함수들 역시 “long-form” 형식을 가정하고 설계된 경우가 많습니다.

  • 따라서 “wide-form”에서 “long-form”으로 왔다갔다 할 수 있어야 합니다.

0단계: tidy data (깔끔한 데이터) 준비하기

1단계: 어떤 변수를 사용할 것인가? (mapping)

  • aes() 함수 안에 x축과 y축에 해당하는 변수를 지정

  • data %>% ggplot(mapping = aes(x=x, y=y, color=z, …))

2단계: 어떤 모양을 사용할 것인가? (geometries)

  • 점? 선? 막대? (https://github.com/rstudio/cheatsheets/blob/main/data-visualization.pdf)

  • geon_point(), geom_line(), …

3단계: 스케일, 축 범위, 범례 등

  • scale_x_log10(), scale_x_date(), scale_x_datetime(), …

  • x_lim, y_lim, scale_y, scale_alpha, scale_color, scale_size, …

ggplot2 - 1. Mapping

library(gapminder)
library(ggplot2)

# ggplot() 함수를 사용합니다.
# mapping = aes(x=x, y=y, ...)로 x축과 y축 등을 지정합니다.
gapminder %>% 
  ggplot(mapping=aes(x=gdpPercap, y=lifeExp))

ggplot2 - 2. Geom

library(gapminder)
library(ggplot2)

# geom_*() 함수를 사용해서 원하는 모양의 그래프를 그릴 수 있습니다.
# ggplot() 함수 다음에 이어지는 함수는 +를 통해 연결합니다.
gapminder %>% 
  ggplot(mapping=aes(x=gdpPercap, y=lifeExp)) +
  geom_point()

ggplot2 - 3. Scale

library(gapminder)
library(ggplot2)

# gdpPercap에 outlier가 존재하기 때문에 두 변수 사이의 관계를 보기 힘듭니다. (1950~1970년대 쿠웨이트)
# 이 outlier들을 제거할 수도 있지만, 여기서는 gdpPercap을 log 변환 시킨 후 x축에 나타내면 관계를 좀 더 잘 볼 수 있습니다.
gapminder %>% 
  ggplot(mapping=aes(x=gdpPercap, y=lifeExp)) +
  geom_point(alpha=0.5) + 
  scale_x_log10()

geom_smooth

library(gapminder)
library(ggplot2)

# geom_smooth()는 추세선을 보여줍니다.
# "lm"은 linear model을 의미합니다.
gapminder %>% 
  ggplot(mapping=aes(x=gdpPercap, y=lifeExp)) +
  geom_point(alpha=0.5) + 
  scale_x_log10() +
  geom_smooth(method="lm")

막대 그래프 - 기본

library(gapminder)
library(ggplot2)

# geom_col()은 막대 그래프를 생성합니다.
# scale_x_continuous로 관측연도를 x축에 나타냈습니다.
gapminder %>% 
  filter(country=="Korea, Rep.") %>% 
  ggplot(mapping=aes(x=year, y=gdpPercap)) + 
  geom_col() + 
  scale_x_continuous(breaks=seq(1952, 2007, 5))

막대 그래프 - 그룹 및 텍스트

library(gapminder)
library(ggplot2)

# 국가 별로 막대의 색을 다르게 하기 위해 aes(..., fill=country)을 넣어줍니다.
# geom_col(position="dodge")를 넣어서 두 그룹의 1인당 GDP가 다른 막대에 표시되도록 합니다.
# 막대 위에 텍스트가 표시되도록 geom_col() 다음에 geom_text()를 넣어줍니다.

gapminder %>% 
  filter(country=="Korea, Dem. Rep."|country=="Korea, Rep.") %>%
  ggplot(mapping=aes(x=year, y=gdpPercap, fill=country)) +
  geom_col(position="dodge") +
  geom_text(aes(label=round(gdpPercap)),
            color="white",
            size=5,
            vjust=1.5,
            position=position_dodge(4.5)) +
  scale_x_continuous(breaks=seq(1952, 2007, 5)) +
  scale_fill_brewer(name="국가", labels = c("북한", "대한민국"), palette="Pastel1") +
  labs(
    x="연도",
    y="1인당 GDP",
    title="대한민국과 북한의 1인당 GDP 추이"
  ) +
  theme(
    axis.ticks.x=element_blank(),
    axis.ticks.y=element_blank()
  )

선 그래프 - 기본

library(gapminder)

# geom_line()은 선 그래프를 생성합니다.
# geom_line() 뒤에 geom_point()를 더해 점을 더할 수 있습니다.

  geom_line() +

선 그래프 - 그룹

library(ggplot2)
library(gapminder)

# 국가 별로 선의 색을 다르게 하기 위해 aes() 안에 color=country를 넣어줍니다. 
# geom_line() 내부의 linewidth로 선의 두께를 조정할 수 있습니다.
gapminder %>% 
  filter(country %in% c("China", "Japan", "Korea, Rep.")) %>% 
  ggplot(mapping=aes(x=year, y=lifeExp, color=country)) +

  geom_point() +

  labs(
    x="연도",
    y="평균수명",
    title="한중일 평균수명 추이"
  )

선 그래프 - 선 및 포인트 모양

library(ggplot2)
library(gapminder)

# geom_line() 내부의 linetype으로 선의 모양을 조정할 수 있습니다.
# geom_point() 내부의 shape으로 점의 모양을 조정할 수 있습니다.
gapminder %>% 
  filter(country %in% c("China", "Japan", "Korea, Rep.")) %>% 
  ggplot(mapping=aes(x=year, y=lifeExp, color=country)) +
  geom_line(linetype = "dashed", linewidth=1) + 
  geom_point(shape = 22, size = 2.5, fill = "white") +
  scale_x_continuous(breaks=seq(1952, 2007, 5)) +
  labs(
    x="연도",
    y="평균수명",
    title="한중일 평균수명 추이"
  )

theme()을 조정한 선 그래프 예시

library(ggplot2)
library(gapminder)

gapminder %>% 
  filter(country %in% c("China", "Japan", "Korea, Rep.")) %>% 
  ggplot(mapping=aes(x=year, y=lifeExp, color=country)) +
  geom_line(linetype = "dashed", linewidth=1) + 
  geom_point(shape = 22, size = 2.5, fill = "white") +
  scale_y_continuous(limits=c(40, 90)) +
  scale_x_continuous(breaks=seq(1952, 2007, 5)) +
  scale_color_brewer(name="국가", labels = c("중국", "일본", "대한민국"), palette="Set1") +
  labs(
    x="연도",
    y="평균수명",
    title="한중일 평균수명 추이"
  ) +
  theme(
    plot.title = element_text(colour="black",
                              size = 28,
                              face="bold"),
    axis.ticks.x=element_blank(),
    axis.ticks.y=element_blank(),
    axis.title.x = element_text(color="black",
                                size=15),
    axis.title.y = element_text(color="black",
                                size=15),
    legend.position="top",
    legend.background=element_blank(),
    legend.title = element_blank(),
    legend.box = "horizontal",
    legend.text=element_text(size=15),
    panel.grid.minor = element_blank(),
    panel.grid.major.y = element_line(linetype="dashed", color = "gray50", size = 0.5),
    #panel.grid.major.x = element_line(linetype="dotted", color = "gray50", size = 0.5),
    panel.background = element_blank(),
    line = element_blank()
  )
